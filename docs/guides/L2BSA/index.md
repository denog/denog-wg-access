# L2BSA

Layer 2 Bitstream Access (L2BSA) beschreibt eine standardisierte Schnittstelle, 
über die ein Infrastruktur-Anbieter seine Zugangsnetzinfrastruktur anderen 
Dienstanbietern zur Verfügung stellt. 

In Deutschland ist L2BSA von der Bundesnetzagentur (BNetzA) vorgeschrieben, 
der Regulierungsbehörde für Strom, Gas, Telekommunikation, Post und Schiene – einer Bundesbehörde des Bundesministeriums für Wirtschaft und Energie mit Sitz in Bonn.

Die aktuielle L2BSA-Spezifikation wurde vom NGA-Forum definiert.

https://www.bundesnetzagentur.de/DE/Fachthemen/Telekommunikation/Breitband/NGA-Forum/start.html
